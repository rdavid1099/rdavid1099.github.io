Trelora Signage Application

During my internship at the Denver-based real estate company Trelora, I was tasked to implement a new feature into the company's existing application handling its signage across the office. Using React.js and Ruby on Rails, I created additional functionality, allowing admins to create custom slides to be displayed on the company's monitors. The application automatically generates and previews four different types of displays, depending on the text the admin enters. All of the design is rendered using Sass, with the preview being generated using React.js.

NewZ Application

"Connecting local news producers with the communities they serve." That's my goal of NewZ. After spending five years in the news industry, I discovered how difficult it was for a producer to connect with and inform a community. NewZ utilizes the New York Times API to generate a list of top stories. Viewers are then able to "pitch" those stories to producers who are in their viewing area. Each pitch can then be commented on and liked/ disliked, sparking conversations between the two groups. Under the hood, I deployed this site to DigitalOcean using Capistrano and have set up CronJobs hitting the New York Times API and populating the database.

Acquire-A-Hire

Nearing the end of the Turing curriculum, we were placed in groups of four and tasked with "cloning" a popular website. Our group created a clone of Thumbtack, the popular app connecting professionals with buyers. Along with the basic functionalities of connecting users of different roles with each other, including messaging, we also implemented our own versioned API and handrolled our own OAuth2.0. I handled the DevOps on this project, deploying the application to DigitalOcean using Capistrano.
